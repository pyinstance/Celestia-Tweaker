# Celestia

A clean, focused Windows optimisation suite built for clarity, control and real results.

Celestia is designed to remove the noise from system tuning. No fake tweaks, no cluttered UI — just a minimal, glass-styled interface with meaningful controls and visible feedback.

---

## Overview

Celestia provides a structured optimisation workflow:

- Scan your system  
- Review detected issues  
- Apply controlled tweaks  
- Verify the results  

Every action is intentional. You always know what changed and why.

---

## Features

### Dashboard
- System overview at a glance  
- Live activity feed  
- CPU / RAM monitoring  
- Quick optimisation controls  

### Process Manager
- Grouped processes (no clutter like svchost spam)  
- Priority control  
- Efficiency modes  
- Safe process termination  

### Tools
- Clean temporary files  
- Flush DNS  
- Restart Explorer  
- Startup app management  
- Restore point creation  
- Junk scanning  

### Optimisation System
- Profile-based tuning (Balanced / Performance / Extreme)  
- Controlled batch application  
- Verification tracking  

### Profile
- Optimisation history  
- Active tweaks  
- Performance score  
- System snapshot  

---

## Design

Celestia uses a minimal black / glass UI with subtle accenting.

The focus is:
- clean layout  
- consistent spacing  
- soft depth and layering  
- smooth micro-interactions  

Nothing flashy. Everything intentional.

---

## Philosophy

Celestia is built on a few simple rules:

- Real controls only  
- Clear feedback always  
- Minimal UI, maximum clarity  
- No fake performance claims  

If something doesn’t provide value, it doesn’t exist.

## Installation

Clone the repository:

git clone https://github.com/yourusername/celestia.git  
cd celestia  

Install dependencies:

npm install  

Run the app:

npm run start  

---

## Build

To build a production version:

npm run build
